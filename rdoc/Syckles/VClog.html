VClog

VClog plugin. Automatically generate changelogs from SCM commits messages.

Attributes

version[RW]

Current version of project.

layout[RW]

Root level changelog layout (gnu or rel). Default is gnu.

format[RW]

Changelog format (html, xml or txt). Default is html. This only applies to the changelog saved to the log/ directory.

typed[RW]

If set to true will include only typed commits. (types are set by adding “[type]” at end of commit messages.

output[RW]

File name to store root level ChangeLog. If not given, it will look for a CHANGES or CHANGELOG file. If that is not found that a root level ChangeLog will not be generated. But a log/changelog file will still be created.

    def document_public_changelog
      if output
        file = project.root + output
      else
        file = project.root.glob_first("{CHANGES,CHANGELOG}{.txt,}", :casefold)
      end

      if file
        #file = (project.root + file).to_s
        if dryrun?
          report "vclog > #{file}"
        else
          public_changelog = typed ? changelog.typed : changelog
          if layout == 'rel'
            changed = public_changelog.save(file.to_s, :rel, metadata.version, metadata.release)
          else
            changed = public_changelog.save(file.to_s, :gnu)
          end
          if changed
            report "Updated #{relative_from_root(file)}"
          else
            report "#{relative_from_root(file)} is current"
          end
        end
      end
    end
